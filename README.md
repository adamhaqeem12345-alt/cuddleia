# Cuddleia - Cozy Digital Goods

Cuddleia is a cozy digital storefront designed to bring warmth, beauty, and serenity to your digital life through thoughtfully designed Islamic booklets and wallpapers.

## Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS & ShadCN UI
- **AI Integration:** Firebase Genkit
- **Payments:** PayPal
- **Notifications:** Telegram Bot API & Google Sheets (Sales Logging)

## Setup
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Configure environment variables in `.env`.
4. Run the development server: `npm run dev`.
