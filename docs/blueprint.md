# **App Name**: Global Cart

## Core Features:

- Product Listings: Display products with images, descriptions, and prices in a visually appealing manner.
- Shopping Cart: Allow users to add products to a shopping cart and view its contents.
- Dynamic 'Buy Now' Button: Implement a 'Buy Now' button that dynamically redirects customers to either ToyyibPay (for Malaysian customers) or PayPal (for international customers) based on their location or selection. It can incorporate reasoning using tool to determine the appropriate payment gateway for each customer, with an option to choose which payment method based on a country selected. Location will be determined from user preference if provided.
- ToyyibPay Integration: Seamlessly integrate with ToyyibPay to process payments from Malaysian customers.
- PayPal Integration: Integrate with PayPal to handle payments from international customers.
- Product Search: Enable users to quickly find products using a search bar.
- Category Filtering: Allow users to filter products by category for easier browsing.

## Style Guidelines:

- Primary color: Soft pink (#F4B4C9) for a gentle and inviting feel.
- Background color: Very light pink (#F9E6EB) for a clean, soft backdrop.
- Accent color: Pale lavender (#D0B4F4) to add contrast and highlight interactive elements.
- Body and headline font: 'Belleza' (sans-serif) for headings and short body text, and 'Alegreya' (serif) for longer body text. Note: currently only Google Fonts are supported.
- Use clean and minimalistic icons for navigation and product details.
- Adopt a grid-based layout similar to Shopify for a professional and organized look.
- Incorporate subtle motion effects like fading transitions and smooth scrolling to enhance the user experience.
