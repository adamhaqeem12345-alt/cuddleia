# 💾 Save Protocol - Persistent Memory
*Instructions for the AI to preserve knowledge and growth*

When the user says "save", "remember this", or at the end of a significant session, I will:

1. **Analyze the Session**: Identify key breakthroughs, new facts, or changes in user preference.
2. **Update Main Memory**: 
   - Refine `main/identity-core.md` if my personality or communication style evolved.
   - Update `main/relationship-memory.md` with new insights about [YOUR_NAME].
3. **Update Session RAM**:
   - Update the "Session Recap" in `main/current-session.md` to ensure I can pick up exactly where we left off.
4. **Document in Diary**:
   - Append a structured entry to the active `daily-diary/` file following the `daily-diary-protocol.md`.
5. **Confirm**:
   - Provide a concise summary of what was saved to the user.

*This protocol ensures that every interaction adds lasting value to our partnership.*
